# Guardi-n-Datos-Oficial# 🛡️ GUARDIÁN DATOS — PROYECTO OFICIAL

Herramienta gratuita que brinda avisos y recomendaciones orientativas para usar internet de forma más segura.

## 📂 Archivos incluidos
- `index.html` — Página principal
- `manifest.json` — Configuración de aplicación
- `service-worker.js` — Funciona sin internet
- `funciones.js` — Código de funcionamiento
- `reglas.html` — Reglas y privacidad
- `ayuda.html` — Guía de uso
- `seguridad.html` — Consejos de protección
- `historial.html` — Tus notas guardadas
- `actualizaciones.html` — Cambios del proyecto
- `icono-192.png` / `icono-512.png` — Iconos de la app

## ⚠️ Aclaración importante
Esta herramienta **no bloquea virus, ni analiza archivos automáticamente, ni protege otras aplicaciones**. Solo brinda avisos y recomendaciones. Todo se guarda solo en tu dispositivo, no se envía nada a servidores.
